# deviceChangePin

## Change device pin

Change the device pin code.

```typescript
const result = await HardwareSDK.deviceChangePin(connectId, params);
```

## Params

****[**Optional common params**](../common-params.md)


```typescript

    payload: {

    }
}
```

Error

```typescript

    payload: {

        code: number // error code
    }
}
```
